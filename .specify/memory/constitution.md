# WARScribe-Parser Constitution

**Version**: 1.0.0 | **Ratified**: 2026-02-01

## Core Principles

### I. Specification Fidelity
Parsing MUST conform to the WARScribe specification exactly.

### II. Error Clarity
Parse errors MUST identify line, column, and expected token.

### III. Non-Destructive
Round-trip parsing MUST preserve all original data.
